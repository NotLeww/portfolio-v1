To add your profile picture:

1. Save your photo as "profile.jpg" in the same folder as index.html (C:\xampp\htdocs\PortfolioV1\)
2. Make sure it's a square image (recommended 300x300 pixels or larger)
3. The code will automatically display it as a circular profile picture

If you want a different filename, update the src in index.html from "profile.jpg" to your filename.

For best results:
- Use a high-quality photo
- Crop it to square aspect ratio
- Good lighting and professional appearance
